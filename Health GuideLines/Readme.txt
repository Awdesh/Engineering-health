Instructions-:

C# Console Application. Add .csproj in C# solution and run via normal process.

Program provides "4" option to choose.
Please press number to choose Option from below-:
 1.) Blood Pressure
2.) HDL
3.) LDL
4.)BMI

On Choosing any option it'll give range detail of selected option-:
e.g if you choose 1.) 
below details will shows up.

Less than 120/80,  NORMALormal
 120-139/80-89 Prehypertension
 140-159/90-99 Stage 1 Hypertension
 160 and above/100 and above Stage 2 Hypertension
