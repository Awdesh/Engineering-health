Instructions-:

Unlike other apps this app tells user how many mins in a day are left for him/her to walk to stay active.

how to call-:
// below will prompt user to enter time in mins that he thinks he had already walked.
// After that API will tell you how many more mins you need to walk.
FitBit.getRemainingTimeNeedToWalk();


You'll see time in mins need to maintain active lifestyle with an inspirational message attached to it!! :)
