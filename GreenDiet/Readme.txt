Instructions-:

node.js applicaiton which uses browserify to load modules in the browser
In order to run this module one should have node and npm installed then.
do below which will install browserify globally.

npm install browserify -g

Once browserify is installed the run below command-:

browserify GreenDiet.js -o bundle.js

above will generate bundle.js file which we can use inside any html file to see o/p inn browser.

Appliction-:

Require js application that counts person intake of green food.

Program asks person to record his/ her daily green intake.
Tells his/her last week green consumption 

Prompts a message to ask his age and then tells how much more needed.
